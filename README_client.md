# JAS Project - Інструкція Користувача (Музей)

Вітаємо у додатку "Draw A Circle"! Цей документ містить всю необхідну інформацію для використання екранів/планшетів в музеї, включаючи приховані функції та налаштування.

---

## 🖥 Керування на Windows Інтерактивних Дошках / ПК
У додатку налаштовано спеціальну підтримку великих екранів:
*   **Масштабування:** Застосунок автоматично підлаштовує розмір інтерфейсу у пропорціях 16:9, тому на надвеликих дисплеях елементи виглядають чітко та пропорційно (віртуальна розмітка 1920x1080).
*   **Вихід з програми:** Оскільки застосунок запускається в ексклюзивному повноекранному режимі, у правому верхньому куті є напівпрозорий хрестик **(Х)** для безпечного виходу з нього.

## 🎮 Функціонал

### 1. Малювання Кола та Алгоритм Оцінювання
*   **Мета**: Намалювати ідеальне коло за відведений час.
*   **Оцінка**: За замовчуванням додаток використовує метод **RMSE (Середньоквадратичне відхилення)**. 
    - Програма знаходить ідеальний математичний центр вашого малюнку.
    - Вираховує "ідеальний радіус" на основі всіх ваших точок.
    - Оцінює, наскільки кожна намальована точка відхиляється від ідеального кола (чим менше відхилення – тим вищий %).
    - Також алгоритм перевіряє "замкненість" фігури (чи з'єднані початок і кінець).
*   *(Примітка: У коді також закладено інші методи, такі як Roundness (за площею) або Hausdorff, але базовий RMSE є найсправедливішим для відвідувачів).*

### 2. Таблиця Рекордів
*   Топ-5 найкращих спроб відображаються у таблиці рекордів.
*   Скидання рекордів доступне в Адмін-панелі.

### 3. Багатомовність
*   Підтримка **UA** (Українська), **EN** (Англійська), **DE** (Німецька).
*   Мова перемикається в Налаштуваннях.

## 🔒 Режим "Кіоск" (Захист від дітей/відвідувачів)

Щоб діти та відвідувачі музею не могли згорнути гру, зайти в налаштування системи або вимкнути планшет, необхідно увімкнути режим "Кіоск". Робиться це на рівні самої операційної системи (додаток не може сам заблокувати пристрій):

### Для планшетів Android (Закріплення екрану):
1. Зайдіть у налаштування самого планшету -> **Безпека** (або "Біометрія і безпека" / "Додаткові налаштування").
2. Знайдіть функцію **Закріплення екрану (Screen Pinning)** і увімкніть її.
3. Відкрийте наш додаток "Draw A Circle".
4. Натисніть кнопку "Останні програми" (три смужки внизу) і натисніть на іконку нашого додатку зверху.
5. Виберіть **Закріпити (Pin)**. Тепер вийти з гри можна лише ввівши системний пароль планшету.

### Для iPad (Гід доступу / Guided Access):
1. Налаштування -> **Доступність (Accessibility)** -> **Гід доступу (Guided Access)** -> Увімкнути.
2. Встановіть пароль (Passcode Settings).
3. Відкрийте наш додаток "Draw A Circle".
4. Натисніть кнопку Home тричі (або кнопку блокування тричі на нових iPad).
5. Натисніть **Start**, щоб заблокувати айпад на цій грі.

### Для електронних дошок на Windows 10/11 (Оскільки це класичний .exe файл):
*Стандартне меню "Assigned Access" у Windows показує лише додатки з Microsoft Store. Оскільки наша програма є класичною програмою для ПК, ми використовуємо "Альтернативний кіоск":*

**Варіант А (Найпростіший: Автозавантаження + Без клавіатури):**
1. Створіть локальний обліковий запис у Windows (наприклад, "Відвідувачі").
2. Увійдіть у нього, натисніть `Win + R`, введіть `shell:startup` і натисніть Enter.
3. Скопіюйте туди ярлик гри "Jas Project". (Гра запускатиметься сама при старті дошки).
4. Оскільки гра розгортається ексклюзивно на весь поверх усього і не має віконних рамок, гравець бачить лише її. 
5. Для безпеки просто фізично відключіть клавіатуру/мишку (або заблокуйте жести Windows), щоб керування йшло лише через малювання по екрану.

**Варіант Б (Жорсткий кіоск через заміну Windows Shell):**
Цей метод повністю вимикає Робочий стіл і залишає лише гру. *(Доступно на Windows Pro/Enterprise)*:
1. На комп'ютері натисніть `Win + R`, введіть `gpedit.msc` і натисніть Enter.
2. Зліва перейдіть у: **Конфігурація користувача -> Адміністративні шаблони -> Система**.
3. Справа знайдіть пункт **Спеціальний інтерфейс користувача (Custom User Interface)**.
4. Увімкніть його (Enabled) та в полі "Назва файлу інтерфейсу" впишіть шлях до гри: `C:\Program Files (x86)\JasProject\jas_project.exe` (або ваш шлях встановлення).
5. Тепер при вході в систему Windows взагалі не завантажуватиме вікна та меню "Пуск" — буде миттєво відкриватись лише ваша гра!

## 🥚 Пасхалки (Секрети)

У додатку приховано 6 секретних функцій ("пасхалок") для розваги відвідувачів та персоналу.

### 1. Фото Команди 📸
Показує фото розробників додатку.
*   **Як відкрити**:
    1. Зайдіть в **Налаштування** -> **Вхід Адміністратора**.
    2. Введіть код: **`TEAM`**
    3. Натисніть OK.

### 2. Кіт Шредінгера 🐱
Перевірка гіроскопу (працює тільки на мобільних пристроях/планшетах).
*   **Як відкрити**: Сильно **потрясіть** планшет.
*   **Ефект**: З'явиться повідомлення від кота.

### 3. Чорна Діра ⚫
Імітація збою системи (/ ділення на нуль).
*   **Як відкрити**: Введіть код **`1/0`** у полі паролю адміністратора.
*   **Ефект**: Екран "ламається" та темніє на 3 секунди.

### 4. Ретро Режим (Сепія) 📜
Змінює вигляд додатку на старовинний стиль.
*   **Як відкрити**: Введіть код **`OLD`** або **`SCHOOL`** у полі паролю адміністратора.
*   **Ефект**: Весь інтерфейс стає коричнево-жовтим. Щоб вимкнути, введіть код ще раз.

### 5. Режим Матриці (Pi Code) 🥧
Змінює вигляд на стиль "Матриці" + конфетті.
*   **Як відкрити**: На екрані "Про проект" (About) натисніть на логотипи у ритмі числа Пі (3 - 1 - 4):
    1. Натисніть **3 рази** (пауза 1 сек).
    2. Натисніть **1 раз** (пауза 1 сек).
    3. Натисніть **4 рази**.
*   **Ефект**: Зелений фільтр та салют.

### 6. Outex Bicep 💪
Мотивація від Outex.
*   **Як відкрити**: На екрані "Про проект" натисніть на текст опису (білий текст унизу) **5 разів** поспіль.

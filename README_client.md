# JAS Project — Інструкція Користувача (Музей / Кіоск)

> **Версія 1.0.0** | Квітень 2026 | Windows Desktop Edition

Вітаємо у застосунку **"Draw A Circle"**! Це інтерактивна гра для сенсорних планшетів та електронних дошок у музеї. Документ містить усю потрібну інформацію для операторів, персоналу та адміністраторів.

---

## 🆕 Що нового у версії 1.0.0

- 🖥 **Повна адаптація під Windows-планшети та великі екрани**: Кнопки, шрифти та елементи інтерфейсу повністю переосмислені та перебудовані під сенсорне управління — не масштабування мобільного, а нативний Desktop-дизайн.

## 🖥 Керування на Windows-планшеті

Застосунок є нативним повноекранним Windows-додатком, оптимізованим для **сенсорного управління** на 17" та більших дисплеях:

- **Масштабування**: Інтерфейс автоматично адаптується до будь-якої роздільної здатності (базова сітка 1920×1080).
- **Вихід з програми**: Кнопка **"ВИЙТИ З ГРИ"** (червона кнопка) доступна в **Адмін-панелі → Налаштування**. Гравці самостійно вийти не можуть.
- **Без мишки**: Застосунок повністю керується пальцем або стилусом.

Для доступу до налаштувань (зміна складності, скидання рекордів, вихід):

## 🎮 Функціонал

### 1. Малювання Кола та Алгоритм Оцінювання
- **Мета**: Намалювати ідеальне коло одним безперервним рухом пальця.
- **Алгоритми оцінювання** (вибираються в Адмін-панелі):
  | Метод | Опис |
  |---|---|
  | **RMSE** (за замовчуванням) | Вимірює відхилення точок від середнього радіуса |
  | **Roundness** | Порівнює площу та периметр фігури |
  | **Combined** | Комбінує roundness + плавність, штрафує за кути |
  | **Hausdorff** | Вимірює розкид відстаней від центру |

### 2. Таблиця Рекордів
- Топ результати зберігаються між сесіями.
- Скидання рекордів доступне в **Адмін-панелі → Data & Leaderboard → Clear Leaders**.

### 3. Багатомовність
- Підтримка **UA** (Українська), **EN** (Англійська), **DE** (Німецька).
- Перемикається в **Налаштуваннях → Language**.

## 🔒 Режим "Кіоск" (Захист від несанкціонованого виходу)

### Для Windows-планшетів/дошок (рекомендований варіант):

**Варіант А — Найпростіший (Автозавантаження):**
1. Створіть локальний обліковий запис Windows (наприклад, "Відвідувач").
2. Натисніть `Win + R`, введіть `shell:startup`, натисніть Enter.
3. Додайте ярлик `jas_project.exe` до папки автозавантаження.
4. Застосунок стартуватиме автоматично при вмиканні планшету.

**Варіант Б — Жорсткий кіоск (Shell Replacement):**
1. `Win + R` → `gpedit.msc` → **Конфігурація користувача → Адміністративні шаблони → Система → Спеціальний інтерфейс користувача**.
2. Активуйте й вкажіть шлях до `jas_project.exe`. Windows завантажуватиме лише гру замість робочого столу.

### Для Android-планшетів (резервний варіант):
1. **Налаштування → Безпека → Закріплення екрану**.
2. Відкрийте гру → Останні програми → іконка гри → **Закріпити**.

## 🥚 Пасхалки (Секрети)

### 1. 📸 Фото Команди
- Налаштування → Вхід адміна → код **`TEAM`**

### 2. ⚫ Чорна Діра
- Налаштування → Вхід адміна → код **`1/0`**
- Ефект: екран "ламається" на 3 секунди.

### 3. 📜 Ретро Режим (Сепія)
- Налаштування → Вхід адміна → код **`OLD`** або **`SCHOOL`**
- Інтерфейс стає коричнево-жовтим. Повторний код — вимикає.

### 4. 🥧 Режим Матриці (Pi Code)
- На екрані "Про проект" натисніть на логотип у ритмі: **3 рази** → пауза → **1 раз** → пауза → **4 рази**.
- Ефект: зелений фільтр + конфетті.

### 5. 💪 Outex Bicep
- На екрані "Про проект" натисніть на білий текст опису **5 разів** поспіль.

### 6. 🐱 Кіт Шредінгера *(тільки на Android)*
- Сильно **потрясіть** планшет.
